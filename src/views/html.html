O que é HTML?

HTML é uma linguagem de marcação que serve como uma estrutura para um projeto web, seria o "esqueleto" do projeto.

Elementos ou TAGS

É aquele objeto que podemos identificar dentro de um documento HTML, fica entre os símbolos: <modelo> com seu fechamento utilizando uma " / " dessa forma: </modelo> para ver algumas tags e suas funcionalidades
