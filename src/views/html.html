HTML Básico

HTML é uma linguagem de marcação de Hipertexto que serve como uma estrutura para um projeto web, seria o "esqueleto" do projeto. Os arquivos em HTML podem ser interpretados por navegadores.

O que é HTML?

HTML é uma linguagem de marcação que é usada para estruturar o conteúdo do seu documento. Você utiliza vários elementos/tags para definir, organizar ou separar várias partes do seu projeto.

Elementos ou TAGS

Os elementos no HTML funcionam como termos-chave que instruem o navegador sobre a formatação e apresentação do conteúdo. Essas marcações são sempre encapsuladas por símbolos de chevron (<>), e dentro delas, encontram-se atributos que definem as propriedades dos elementos, orientando a construção do documento HTML e especificando as características dos componentes. O navegador, ao interpretar o documento, segue uma leitura de cima para baixo e da esquerda para a direita.
Segue o exemplo: <modelo> e o fechamento se dá usando uma "/" dessa forma: </modelo>

para ver as principais tags e suas funcionalidades.
<!DOCTYPE html> - Instrução para o navegador entender em qual versão do HTML a página está escrita.
<html> - Elemento raíz do documento HTML
<head> - Fornece informações essenciais, conhecidas como metadados, sobre o documento.
<body> - Representa o conteúdo do documento HTML.
<a> - Também chamada de elemento âncora, é utilizada para criar links em páginas web.
<br> - É usada como quebra-de-linha
<div> - Atua para agrupamento, proporcionando a organização e o agrupamento de conteúdo em um fluxo.
<h1> <-h6> - Representam seis níveis de título de seção.
<main> - Define o conteúdo principal dentro do <body> em seu documento HTML.
<nav> - Representa uma seção de uma página que aponta para outras páginas ou para outras áreas da página.
<title> - Estabelece o título do documento, exibido na barra de título de um navegador ou na aba da página.
<p> - Representa um parágrafo.
